Depure o código diretamente do editor. Inicie ou anexe seus aplicativos em execução e depure com pontos de interrupção, pilhas de chamadas e um console interativo.

Comandos Git integrados.
Trabalhar com Git e outros provedores de SCM nunca foi tão fácil. Revise diferenças, prepare arquivos e faça commits diretamente no editor. Push e pull de qualquer serviço SCM hospedado.

Extensível e personalizável.
Quer ainda mais recursos? Instale extensões para adicionar novos idiomas, temas, depuradores e para conectar-se a serviços adicionais. As extensões são executadas em processos separados, garantindo que não deixarão seu editor lento. Saiba mais sobre extensões.
